## 📌 About the Project

Bruno Portfolio is a responsive and visually engaging website built to present my work in a clean and professional way. It includes smooth animations, interactive UI elements, and project sections with live demo links.

This project reflects my ability to build real-world web interfaces and user experiences.

- ⚡ Smooth animations and transitions
- 🧩 Interactive project cards
- 🔗 Live demo links to projects
